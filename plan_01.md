React와 Vite를 기반으로 '전자칠판 자동 홍보 블로그 생성기' MVP 웹사이트를 만들어 줘. 

1. 프론트엔드 (UI 화면):
- 사용자가 '타겟 고객(예: 학원장, 스타트업 대표)'과 '메인 키워드(예: 스마트 오피스 구축)'를 입력할 수 있는 텍스트 입력창을 만들어 줘.
- '네이버 블로그 포스팅 생성하기' 버튼을 만들어 줘.
- 결과물이 출력되는 텍스트 박스와, 작성된 글을 클립보드에 바로 복사할 수 있는 '복사하기' 버튼을 추가해 줘. 깔끔하고 모던한 UI로 디자인해 줘.

2. 백엔드 (서버리스 & AI API 연동):
- Cloudflare Pages Functions(워커)를 사용하여 백엔드 API를 구성해 줘. 
- OpenAI API(GPT-4o-mini 모델 권장)를 연동해서, 사용자가 입력한 정보를 바탕으로 가독성이 높고 SEO에 최적화된 네이버 블로그 형태의 정보성 홍보 글을 생성하는 프롬프트 로직을 작성해 줘.
- 글의 마지막 부분에는 반드시 자연스럽게 전자칠판 제휴 링크(https://2026-nexo-polic-y-fund.netlify.app/?partner_code=johanju)가 포함되도록 AI 프롬프트에 지시해 줘.
- OpenAI API 키는 프론트엔드에 노출되지 않도록 환경 변수(env)로 숨겨서 처리해 줘.

3. 배포:
- 코드가 모두 작성되면, Git Push를 통해 원격 저장소에 올리고 Cloudflare Pages에 배포하는 작업까지 완료해 줘.
